# GrammarSense AI - AI powered Application
A cross-platform mobile application built with Flutter that uses Google Gemini AI to detect and correct grammatical errors in words, sentences, and paragraphs.

---

## Features

- AI-powered grammar detection and correction
- Detailed suggestions explaining each mistake
- Supports words, sentences, and full paragraphs
- Clean and minimal UI with Google Fonts

---

## Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **AI:** Google Gemini API
- **Platforms:** Android · iOS · Web · Windows · macOS · Linux

---

## Getting Started

### Prerequisites

- Flutter SDK `>=3.29.0`
- Dart SDK `^3.7.2`
- Google Gemini API Key — [Get one here](https://aistudio.google.com/app/apikey)

### Setup

```bash
# Clone the repo
git clone https://github.com/mahesh-lute-9/GrammarSense-AI.git
cd GrammarSense-AI

# Install dependencies
flutter pub get

# Add your Gemini API key in the source file
# Then run the app
flutter run
```
